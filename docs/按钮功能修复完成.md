## 🎯 功能说明

### AI 配置的自定义 API 模式

当选择"自定义 API"时：
1. 输入 API Key 和 Base URL
2. 点击"获取模型"按钮
3. 系统会调用 `{Base URL}/v1/models` 获取可用模型
4. 获取成功后在下拉框中显示模型列表
5. 选择模型后保存配置

**注意**：实际 API 调用将在 Phase 5 实现，当前为演示模式。
