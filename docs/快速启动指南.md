# 快速启动指南

## 🚀 如何启动项目

### 前提条件
- ✅ Node.js >= 20
- ✅ Python >= 3.12
- ✅ npm >= 11

---

## 启动步骤

### 1️⃣ 启动后端（FastAPI）

```bash
# 进入后端目录
cd backend

# 启动 FastAPI 服务
python main.py
```

**服务地址**：http://127.0.0.1:8765  
**API 文档**：http://127.0.0.1:8765/docs

---

### 2️⃣ 启动前端（React）

**新开一个终端窗口**：

```bash
# 进入前端目录
cd frontend

# 启动开发服务器
npm run dev
```

**服务地址**：http://localhost:5173

---

## 🛠️ 常见问题

### Q1: 后端启动失败？
**A**: 检查 Python 版本和依赖：
```bash
python --version  # 应为 3.12+
cd backend
pip install -r requirements.txt
```

### Q2: 前端页面空白？
**A**: 检查浏览器控制台是否有错误，确保后端已启动。

### Q3: 端口被占用？
**A**: 修改 `backend/main.py` 的端口：
```python
uvicorn.run("main:app", host="127.0.0.1", port=8766, reload=True)
```

同时修改 `frontend/vite.config.ts` 的代理配置。
