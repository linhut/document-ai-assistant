## 📝 功能说明

### AI 配置的自定义 API 工作流程

1. 选择"自定义 API"
2. 输入 API Key
3. 输入 Base URL（例如：https://api.openai.com）
4. 点击"获取模型"按钮
5. 系统调用 `{Base URL}/v1/models` 获取模型列表
6. 模型列表显示在下拉框中
7. 选择要使用的模型
8. 点击"保存配置"

**当前状态**：演示模式，返回 Mock 数据  
**Phase 5**：将接入真实 API
