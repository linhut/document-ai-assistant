## 新增的功能

### 后端功能
1. ✅ **POST /api/templates/create** - 创建新模板
   - 接收：模板名称、文档类型、描述、图标
   - 自动生成基础规则结构
   - 保存为 YAML 文件

2. ✅ **GET /api/templates/list** - 列出12个模板
   - 显示所有可用模板
   - 标记是否已配置规则

3. ✅ **GET /api/templates/{id}** - 获取模板详情
   - 返回规则内容
   - 显示检查和修复规则数量

### 前端功能
1. ✅ **新增模板按钮**
   - 位置：模板中心页面右上角
   - 点击打开创建对话框

2. ✅ **创建模板对话框**
   - 输入：模板名称
   - 输入：文档类型（英文标识）
   - 输入：描述
   - 输入：图标 Emoji
   - 按钮：取消 / 创建

3. ✅ **自动生成规则**
   - 自动创建基础检查规则
   - 自动创建基础修复规则
   - 用户可后续手动完善

## 使用方法

### 查看所有模板
1. 进入"模板中心"
2. 看到12个模板卡片
3. 点击查看详情

### 新增自定义模板
1. 点击右上角"新增模板"按钮
2. 填写表单：
   - 模板名称：例如 "通报"
   - 文档类型：例如 "circular"
   - 描述：例如 "批评或表扬的公文"
   - 图标：例如 "📣"
3. 点击"创建"
4. 系统自动生成基础规则文件
5. 可在 `rules/official/` 目录手动完善规则
