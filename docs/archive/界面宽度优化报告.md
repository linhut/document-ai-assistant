## 推荐宽度标准

### 桌面应用标准宽度
```css
/* 单列内容（表单、详情） */
max-width: 1200px;

/* 卡片网格（2-4列） */
max-width: 1400px;

/* 列表/表格（多列数据） */
max-width: 1600px;
```

## 设计原则

### 1. 内容类型决定宽度
- **表单页面**：1000-1200px（防止表单过宽）
- **卡片网格**：1400-1600px（展示3-4列卡片）
- **数据列表**：1600px+（充分展示表格）

### 2. 响应式断点
```css
/* 小屏笔记本 */
@media (max-width: 1366px) { max-w-[1200px] }

/* 标准桌面 */
@media (min-width: 1920px) { max-w-[1600px] }

/* 大屏显示器 */
@media (min-width: 2560px) { max-w-[1800px] }
```
