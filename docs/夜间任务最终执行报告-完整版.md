## 🚀 使用指南

### 启动项目

**后端**：
```bash
cd backend
python main.py
```

**前端**：
```bash
cd frontend  
npm run dev
```

**访问**：http://localhost:5173/

1. 选择 Word 文档
2. 选择文档类型
3. 开始检查

5. 点击"全部应用"
