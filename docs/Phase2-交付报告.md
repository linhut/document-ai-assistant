## 🎨 设计亮点

### ✅ 采用的设计
1. **温暖大地色系** - 避免冷蓝色，符合政务场景
2. **专业桌面风格** - 参考 VS Code / Obsidian
3. **固定侧边栏** - 桌面软件标准布局
4. **问题等级可视化** - 左侧色条 + 色彩映射
5. **关于页面** - 启动后可查看软件信息

### ❌ 避免的设计
1. **后台管理风格** - 不使用复杂表格、数据看板
2. **毛玻璃效果** - 过于现代化
3. **冷色调** - 纯蓝色过于冰冷
4. **全屏模态框** - 破坏工作流

## 附录：核心技术特性

### 前端架构
- **状态管理**：React Hooks（未引入 Redux/Zustand）
- **样式方案**：TailwindCSS 4 + CSS 变量
- **组件库**：shadcn/ui（定制主题）
- **路由**：React Router 7.x
- **HTTP 客户端**：axios（带拦截器）

### 后端架构
- **API 框架**：FastAPI（异步）
- **数据模型**：Pydantic（类型验证）
- **ORM**：SQLAlchemy 2.0
- **数据库**：SQLite（本地）
- **规则引擎**：YAML 配置驱动

### 设计模式
- **MVC 分离**：前后端独立
- **组件化**：可复用组件
- **配置驱动**：规则文件 YAML
- **插件化**：AI Provider 抽象接口
