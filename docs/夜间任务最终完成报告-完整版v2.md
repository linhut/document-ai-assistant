## 🚀 使用指南

### 启动项目

**后端**：
```bash
cd backend
python main.py
```

**前端**：
```bash
cd frontend
npm run dev
```

**访问**：http://localhost:5173/

### 完整使用流程
1. **上传文档**
   - 选择 Word 文档
   - 选择文档类型（12种）
   - 点击"开始检查"

2. **查看问题**
   - 自动跳转校审中心
   - 查看所有检查问题
   - 按严重程度筛选

3. **应用修复**
   - 点击"全部应用"
   - 等待修复完成
   - 看到成功提示

4. **下载文档**
   - 点击"下载优化文档"
   - 获得优化后的 Word 文件

5. **新增模板**（可选）
   - 进入模板中心
   - 点击"新增模板"
   - 填写表单创建
