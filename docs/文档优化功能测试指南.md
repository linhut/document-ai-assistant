## 常见问题

### Q1: 上传失败
**原因**：后端服务未启动  
**解决**：
```bash
cd backend
python main.py
```

### Q2: 优化无效果
**原因**：文档类型选择错误  
**解决**：确认选择"会议纪要"类型

### Q3: 字体显示异常
**原因**：系统缺少字体  
**解决**：
- 安装方正小标宋简体
- 安装仿宋_GB2312

### Q4: 下载失败
**原因**：优化文件未生成  
**解决**：
- 检查后端日志
- 确认data/outputs目录存在
