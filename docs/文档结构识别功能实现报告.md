#### 2. 会议纪要特殊识别
```python
meeting_keywords = {

```

#### 3. 智能模式匹配
```python
# 标题识别
- 第一个非空段落
- 包含关键词（通知、请示、报告等）
- 长度限制（<50字）

# 主送机关识别
- 标题后第一段
- 以"各"、"全体"等开头
- 以冒号结尾

# 落款识别
- 文档末尾向前查找
- 包含单位关键词
- 日期前一段
```

#### 1. 文档检查流程
```python
# document_service.py
def check_document(db, doc_id, doc_type):
    model = parse_docx(file_path)
    
    # 新增：结构分析
    analyzer = DocumentStructureAnalyzer()
    structure = analyzer.analyze(model)
    
    # 使用结构信息辅助检查
    rules = load_rules_for_type(doc_type or structure['structure_type'])
```

#### 2. AI优化流程
```python
# ai_provider.py
def analyze_document(model, doc_type):
    # 结构分析
    analyzer = DocumentStructureAnalyzer()
    structure = analyzer.analyze(model)
    
    # 生成AI提示
    prompt = analyzer.generate_ai_prompt(model, structure, rules)
    
    # 调用AI
    response = self.call_ai(prompt)
    return response
```

## 🔧 使用示例

### 基础用法
```python
from core.document.parser import parse_docx
from core.document.structure_analyzer import DocumentStructureAnalyzer

# 解析文档
model = parse_docx("document.docx")

# 结构分析
analyzer = DocumentStructureAnalyzer()
structure = analyzer.analyze(model)

# 查看结果
print(f"标题: 段落{structure['title']}")
print(f"类型: {structure['structure_type']}")
print(f"正文: {structure['body_start']}-{structure['body_end']}")
```

```python
# 生成AI提示
prompt = analyzer.generate_ai_prompt(model, structure, rules)
```
